# AI Nutrition Coach

AI Nutrition Coach is a Colab-based chatbot that provides personalized nutrition guidance using user health and lifestyle inputs. The assistant asks questions about age, height, weight, activity level, goals, health conditions, allergies, and food intake. It then calculates BMI, BMR, TDEE, and a suggested calorie target.

For food analysis, the project connects to the USDA FoodData Central API to retrieve calories, macronutrients, and selected micronutrients. The final report includes body calculations, meal nutrient analysis, rule-based recommendations, and optional AI-generated coaching feedback.

This project is designed as an educational MVP for building a lifestyle and nutrition recommendation agent with API-based nutrient validation.
